A full-featured backend API for a **video streaming and social media platform**. Users can publish videos, create playlists, like content, comment, follow channels, and post tweets. The API supports authentication, authorization, and media uploads via Cloudinary.

## **Features**
- JWT authentication and role-based access  
- Publish, update, delete videos  
- Video views tracking  
- Cloudinary integration for video and thumbnail uploads  
- Playlist creation and management  
- Like/unlike videos, comments, and tweets  
- Add, update, delete, and fetch comments with pagination  
- Tweet creation, update, and deletion  
- Subscribe/unsubscribe to channels  
- Fetch subscribers and subscribed channels  
- Search, sort, and pagination for videos  

---

## **Tech Stack**
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Authentication:** JWT  
- **File Storage:** Cloudinary  
- **Middleware:** Multer (for file uploads), async-handler  
- **Other Packages:** mongoose-aggregate-paginate-v2, bcrypt, jsonwebtoken
